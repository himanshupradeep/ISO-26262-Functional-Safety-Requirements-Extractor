# ISO 26262 Functional Safety Requirements Extractor 🚗📄🔍

A **Streamlit-based tool** that extracts functional safety requirements from ISO 26262 documents. It automatically identifies and highlights:

<img width="1919" height="1069" alt="Screenshot 2025-07-11 224444" src="https://github.com/user-attachments/assets/42c8da4f-0199-438f-b822-f28afee526e9" />

- 🔐 **ASIL Levels (A–D)**
- ⏱ **Fault-Tolerant Time Intervals (FTTI)**
- 🛑 **Safe State Descriptions**

Perfect for **safety engineers**, **compliance teams**, and **automotive developers** working on ISO 26262-compliant systems.

---

## 🚀 Features

✅ Upload ISO 26262 documents (PDF)  
✅ Automatically extract functional safety requirements  
✅ Detect and display ASIL levels  
✅ Identify fault-tolerant timing constraints  
✅ Highlight safe state conditions  
✅ User-friendly Streamlit interface
